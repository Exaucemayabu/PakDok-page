Merci de téléchargé PakDok. Cette version est fournie telle quelle : pensez à sauvegarder vos documents régulièrement.

Android (APK)

Téléchargez et installez manuellement l'APK sur votre téléphone Android. Pour installer, suivez ces étapes :

  1. Ouvrez l’application Paramètres de votre téléphone.
  2. Dans la barre de recherche des paramètres, tapez “Installer des applications inconnues” ou “Sources inconnues”.
  3. Sélectionnez ensuite le navigateur ou gestionnaire de fichiers que vous utilisez pour télécharger l’APK (ex. Chrome, Firefox, Fichiers).
  4. Activez l’option Autoriser à partir de cette source.
  5. Téléchargez l’APK en cliquant sur le bouton ci-dessous.
  6. Ouvrez le fichier téléchargé pour lancer l’installation.
  7. Une fois installé, vous pouvez ouvrir PakDok depuis votre téléphone.
